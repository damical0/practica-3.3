Navegadores web

Un navegador web (en inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web. Interpretando la información de distintos tipos de archivos y sitios web que estos puedan ser vistos.

La funcionabilidad básica de un navegador web es permitir la visuaización de documentos de texto. posiblemente con recursos multimendia incrustados.

Ejemplos de navegadores web

Safari: Safari es un navegador web de código cerrado por Apple Inc. Está disponible àra macOS, IPadOS e iOS

Mozilla Firefox es uin navegador web libre y de cñodigo abierto desarrollado para Linux, Android, iOS, macOS y Microsoft Windows coordinado por la Corporación Mozilla y la Fundación Mozilla

Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
